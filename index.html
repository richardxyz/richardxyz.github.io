开始新的blog之旅

发表于 2016-10-17

最终决定使用使用github pages和hexo搭建blog系统，然而一路并不顺利。

  • markdown
    markdown语法简单易学，看起来像是程序员的工具，在当今移动应用流行的年代，显得极其古老。推荐使用chrome应用stackedit编写，离线亦可以使用。

  • github
    安装github desktop时下载缓慢，安装失败，最后发现是我墙干的好事，amazonaws.com有7%的丢包率，使用静态IP无效，然而第二天却突然好了，比较无奈。解决方案：科学上网，重新拨号，隔一段时间再试。

  • 速度页面
    打开速度过慢，检查后发现hexo使用了google字体，尝试配置为fonts.gmirror.org，稍有改进，但是还是要十来秒。后来在公司打开速度较快，看起来跟网络有一定关系，需要后续观察。
